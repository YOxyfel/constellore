# Constellore 3.0 — The Path Becomes the Game

Constellore 3.0 is the biggest beta update yet. The goal is now unmistakable from the first screen: start with Earth, Water, Fire, and Air, then combine ideas until you create the destination word. Every successful step becomes part of a visible constellation that is yours alone.

## What changed

- **A calmer first session.** First Orbit teaches the board, then Second Orbit teaches a real three-fusion route to Mountain without blocking experimentation. Challenge rules, Daily play, Journeys, Creator's Lab, collections, and community features reveal themselves progressively instead of competing for attention.
- **A universe that remembers.** Explore is now a clearly unranked, target-free sandbox whose discovered-word inventory persists between sessions. Scored missions still reset fairly to Earth, Water, Fire, and Air.
- **A much richer World Graph.** The local beta now contains 1,066 deterministic recipes and 759 playable word records. Its graph audit catalogs 744 concepts, with 686 reachable from the four starters, and covers all 759 reviewed high-intent attempts in the current quality corpus, including natural same-word results such as Water + Water and Fire + Fire.
- **More meaningful destinations.** Ninety deterministic Daily challenge variants, stronger route diversity checks, fewer dead ends, and a much smaller local graph make the game both deeper and faster to load.
- **Guidance that teaches instead of spoiling.** Help now follows one clear ladder: a score-safe Route Signal, an Open-division Compass, a crucial Word Gift, and finally a complete Study reveal. Every level states its scoring effect before you use it.
- **The route stays visible.** Milestones, the Living Atlas, and the result screen show genuine progress through the constellation. Rival Ghost appears only after three real wins and uses verified route length when available.
- **Clear competitive integrity.** Pure, Open, Practice, and Study are visibly distinct. The online service computes ranked outcomes from an exact issued challenge and marks suspicious results provisional; static Pages and itch builds stay honestly unranked.
- **Better beta feedback.** If a logical pair is missing, players can report “Expected this to work” without sending free text or identity. Optional diagnostics are off by default and use only a bounded event vocabulary.
- **Release-grade foundations.** The update adds deterministic release archives, versioned cache metadata, install screenshots and shortcuts, automated mobile/desktop journeys, accessibility scans, performance budgets, public beta policies, and explicit commerce safety gates.

## Still a beta

The downloadable itch build is deterministic local practice: progress stays on this device, and it does not include live AI, accounts, verified leaderboards, ads, or payments. Beta progress may reset as the World Graph evolves. The separate online service is still being hardened and will not accept real-money purchases until its database, fulfillment, legal, support, and operational gates are complete.

If a combination feels obvious and does not work, please use the new in-game report or the recipe report form linked from the Support page. Those reports are how the constellation grows.

Thank you for playing from first light.
