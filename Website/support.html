Beta support

Constellore is made by Oxyfel Games in Bulgaria. Clear reports are valuable, especially when the word graph or mobile controls behave unexpectedly.

Keep secrets private

Never post a Recovery Kit, bearer token, address, payment detail, or other personal information in a GitHub issue, screenshot, Discord message, or recipe report.

Broken or missing combination

Include the two input words, what happened, what you expected, the mode, and whether the build was web, itch.io, or the online beta. Add a screenshot only if it contains no credentials.

Report a recipe →

Bug or accessibility issue

Include your browser/device, the action you attempted, and the result. For accessibility issues, mention the keyboard or assistive technology used.

Report a bug →

Score review

Include the public callsign, challenge, division, approximate completion time, and visible score. Do not include a run token or Recovery Kit.

Request review →

Security concern

Do not publish an exploitable vulnerability, real credential, or player data. Use GitHub’s private security-reporting interface when available; otherwise report only that private contact is required.

Open private report →

Self-service profile controls

Profile contains export and deletion actions. Local practice can also be erased through browser site-data settings. An online Recovery Kit rotates after successful recovery. Oxyfel Games cannot safely recover a pseudonymous profile without proof of control.

Current response boundary

This is a small public beta without a contracted support desk or payment support. Public reports are best-effort and should contain no sensitive information. A tested private support mailbox, documented response targets, escalation, and appeal workflow remain required before commercial launch.
